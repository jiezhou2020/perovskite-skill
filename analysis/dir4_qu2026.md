# Qu 2026 · SPM 钙钛矿表面/界面综述 · 11-field 深读

## 1. 元数据

- **一作 / 通讯**：Z. Qu / S. X.（具体姓名见原文，作者贡献章节）
- **期刊**：EES Solar（RSC，2026 在线，OA）
- **方向归属**：dir4（表征解耦） — **SPM 工具大全的方法学综述**
- **类型**：Review，2022 年后大量新工具的关键集合（pp-KPFM / fast-KPFM / SPVM / s-SNOM / PL-AFM / AFM-IR / PTIR）

## 2. 研究动机 (Why)

钙钛矿光电性能受**纳米级异质性**主导：晶界、点缺陷、组分梯度。而宏观测量（J–V、PL 集成）平均掉这些信息。SPM 给纳米空间分辨，但单一模式不够 — 需要把 SPM 与光学、时间分辨结合。Qu 2026 系统综述了 2022 之后的新组合工具。

## 3. 核心主张 (1 句)

> SPM 已经从"看形貌"演化为"原位 operando 多模态平台"，覆盖**空间从 sub-nm 到 μm、时间从 ps 到分钟**的钙钛矿表征 — 是用户 dir4 评估协议的工具调色盘最权威的来源。

## 4. 方法 (How)

综述。系统覆盖以下技术：

**第 2 节 · SPM 基础**：
- **OBD-AFM**（光束偏转 AFM）— 标准形貌
- **qPlus-AFM** + CO 探针 — 化学键级分辨（Telychko 在 RPP 上展示）
- **STM** — 原子分辨；可观察光照 vs 暗态下表面晶格变化（MAPbBr3 暗态 dimer → 光照 4×2 super-lattice）
- **c-AFM**（导电 AFM）— 局部光电流分布；GB vs 晶内的离子迁移行为差异
- **KPFM** — 表面势 / 接触电势差（CPD）；CPD = 工作函数 - tip 工作函数 + band-bending + surface charge

**KPFM 应用细化**：
- 横向器件（lateral）：电学极化前后 CPD 对比 → 离子迁移可视化（Yuan 2015）
- 偏压下 CPD 极性反转 → GB 被识别为离子迁移路径（Yun 2016）
- **相分离专用**：CH3NH3PbBr0.9I2.1 在晶界处 CPD 显示**向上 band-bending** + 局部正空间电荷 → I⁻ 被吸引到 GB 形成 I-rich 域（Tang 2018）
- 横切面 KPFM：器件多层结构（perovskite/TiO2 等）内置电场可视化；揭示界面 transport bottleneck

**第 3 节 · SPM 扩展**：
- **PL-AFM / SNOM**：sub-50 nm PL 分辨；与共聚焦 PL（200–300 nm）相比；MAPbI3 hot spot 与晶相质量相关
- **AFM-IR / PTIR**：10–20 nm 红外振动光谱分辨；MAPbI3-xClx 中 Cl 分布、退火前后；离子迁移可视化
- **s-SNOM + KPFM 同步**：空间分辨 27 nm（s-SNOM）/ 57 nm（KPFM）；GB 处 IR near-field 增强、表面势 downward bending → 电子积聚
- **SPVM**（surface photovoltage microscopy）：CPD 在调制光下变化 → 直接看光生载流子的局部分离与积聚

**第 4 节 · 时间分辨**：
- **fast-KPFM**：每分钟数张 CPD 图，捕获 ms-s 量级表面势演化；MAPbI3 光照后表面势数分钟才弛豫 → 主要是离子迁移而非光载流子（Garrett 2017）
- **pp-KPFM（pump-probe KPFM）**：光/电脉冲泵浦 + 探针延时；时间分辨 4 ns（电脉冲）— **最快可达 1 ps**（光泵浦，LT-GaAs 验证）
- 双反馈环路设计区分时间平均 CPD 与瞬态 CPD

**第 5 节 · Artifacts**：
- Tip 几何 / 工作函数变化导致 CPD 偏差
- Tip-induced band bending（特别是半导体）
- 离子-电子混合传输使长寿命电势信号被误读为电子结构
- 推荐策略：lift-mode AM-KPFM, FM-KPFM no-feedback, UHV constant-height

## 5. 量化锚点 + 突破

| 锚点 | 数值 |
|---|---|
| qPlus-AFM 分辨 | 化学键级（< 0.5 nm） |
| STM 分辨 | sub-Å（原子级） |
| **PL-AFM 分辨** | **< 50 nm**（相比共聚焦 PL 200–300 nm，5× 提升） |
| AFM-IR / PTIR 分辨 | **10–20 nm**（远超 IR 衍射极限） |
| s-SNOM 分辨 | **27 nm**（vs KPFM 57 nm 同区） |
| CPD 横切面变化 | 155 mV（perovskite vs TiO2 界面）（Bergmann 2015） |
| **fast-KPFM 时间分辨** | 几张图/分钟（ms–s 级） |
| **pp-KPFM 最快时间分辨** | **1 ps**（光泵浦，LT-GaAs；理论极限来自 cantilever 热噪声） |
| MAPbI3 表面势光照后弛豫时间 | **数分钟**（提示离子迁移而非光载流子） |
| GB 处电子密度变化（s-SNOM + KPFM） | 6×10¹⁹ cm⁻³（暗）→ 8×10¹⁹ cm⁻³（光）（CH3NH3PbI3） |

## 6. 解决的问题

- 弥补 Tian 2022 综述对 SPM 时间分辨变体（pp-KPFM、fast-KPFM）和 hybrid 工具（PL-AFM、AFM-IR）的覆盖空白。
- 把"光致 vs 离子迁移 vs 缺陷态"三种贡献用 SPM 多模态拆开 — 这是 dir4 评估协议的核心需求。

## 7. 局限

1. 综述未给出"工具 × 机制（热力学/极化子/载流子梯度/redox）"的正式方阵 — 这正是用户要补的。
2. SPM 主要做表面/近表面，对 bulk 内部 ion redistribution 不直接 — 需要与 cross-section KPFM、TEM 结合。
3. 离子-电子混合传输使 CPD 解读复杂；用户做评估协议时必须明确 artifacts 章节里的分离方法。
4. pp-KPFM 1 ps 时间分辨的实验门槛极高（需要超快激光、cantilever 热噪声压制、探针几何精确控制），非用户实验室能直接复现。

## 8. 可证伪判据

- **若**KPFM 显示晶界 CPD 偏移 + s-SNOM 显示 IR 增强 + PL-AFM 显示晶界 PL 红移 三者同区一致 **→** 支持"晶界主导相分离 + 表面带弯";若三者方向不一致 → 表征系统揭示机制不是单一晶界故事。
- **若**MAPbI3 光照后 CPD 弛豫时间在 ns–μs 量级（pp-KPFM）→ 主导是光载流子；若在 s–min 量级（fast-KPFM）→ 主导是离子迁移。**这是一个用 KPFM 时间尺度直接区分两机制的判据**。
- **若**AFM-IR 显示退火前后 Cl/I 比空间分布显著变化 + KPFM 显示对应区域 CPD 同步变化 → 离子迁移与表面势耦合得到证实。

## 9. 设备改进钩子

- 把 fast-KPFM + pp-KPFM 整合到一台 SPM 平台 → 横跨 ps–min 的时间窗一次扫描覆盖。
- 加入控制气氛舱（湿度、O2、温度）→ 真正 in situ 老化下监测，对接 Frohna 2024 的 ISOS-L-2I 协议。
- AFM-IR 加入光照泵浦同步 → 从静态化学分布升级到 operando 离子迁移跟踪。
- 与同步辐射 nXRF 联用（参考 Frohna toolkit）→ KPFM 表面势 + nXRF 化学分布跨尺度对照。

## 10. 对博士研究的启示

**这是用户 dir4 评估协议工具调色盘的最直接来源**：

1. **三机制 × SPM 工具映射可以从这里直接抽出来**（见本批的 toolkit 综合文件）。
2. **时间分辨是 dir4 的关键差异化锚点** — 用 fast-KPFM/pp-KPFM 可以**直接区分"光载流子主导"还是"离子迁移主导"**，把 Tian 2022 列出的 4 类机制中的"载流子梯度/trap"和"redox"两类用时间窗鉴别开来 → 这是"反向验证"的实验设计金钥匙。
3. **artifacts 章节的内容应该写进研究计划 v3 的"协议失效条件"** — 任何评估协议都必须明确哪些情况下读数不可信。
4. **文末未来方向（spatial/temporal/in situ/multi-modal/ML）**几乎就是博士工作可以直接搬过来的 5 个研究方向骨架。

## 11. 未来方向

- 作者列出：(1) 真正原子级分辨 KPFM；(2) ps 时间分辨；(3) 真正 in situ operando 表征环境；(4) 多模态联用；(5) ML 辅助大数据分析。
- 用户可延伸：把这 5 条与 Tian 2022 的 4 个机制类做矩阵，挑出"哪个机制最缺哪种 capability"；那就是博士工作能切入的最高边际价值点。例如，**redox 机制（I⁻ → I⁰ → I2 vapor）目前缺乏直接在 nm 尺度看到 I2 释放的工具** — pp-KPFM + AFM-IR + 控气氛舱可能正好填这个空缺。
